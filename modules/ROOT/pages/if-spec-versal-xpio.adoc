=== Versal PL XPIO

Versal PL XPIO 信号は、Versal デバイスのプログラマブルロジック側にある XPIO に接続されており、ユーザが利用できるように提供されています。XPIO は、高性能インタフェースと低速の SDR/DDR インタフェースの両方をサポートします。XPHY を使用することで、XPIO はデータストリームのアラインメント、シリアライズ、およびデシリアライズを実現します。対応する I/O 規格は、1.0 V、1.2 V、1.35 V、1.5 V のバンク電圧に対応しています。

SC-OBC Module V1 は、ユーザ向けに 54 本の XPIO 信号を提供します。Versal PL XPIO を使用するには、キャリアボードから VCCO_703 を介してバンク電源を供給する必要があり、ユーザは Versal がサポートするいずれかのバンク電圧を選択できます。
