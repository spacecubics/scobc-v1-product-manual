=== Versal PL HDIO

Versal PL HDIO 信号は、Versal デバイスのプログラマブルロジック側の HDIO に接続されており、ユーザが利用できるように提供されています。HDIO（High-Density I/O）は、低速の SDR および DDR インタフェースをサポートするリソースを含みます。これらの I/O は、1.8 V、2.5 V、3.3 V のバンク電圧に対応した規格をサポートします。

SC-OBC Module V1 に搭載されている Versal デバイスには 22 本の HDIO 信号があり、そのすべてがユーザに開放されています。Versal PL HDIO を使用するためには、キャリアボードから VCCO_302 を介してバンク電源を供給する必要があり、Versal がサポートするいずれかのバンク電圧をユーザが選択できます。
