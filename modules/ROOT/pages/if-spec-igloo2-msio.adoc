=== IGLOO2 MSIO 0/1

IGLOO2 MSIO 0/1 は、IGLOO2 デバイスの MSIO に接続された I/O 信号であり、ユーザが利用できるように提供されています。IGLOO2 の MSIO（Multi-Standard I/O）は高い柔軟性を持つ汎用 I/O で、幅広い I/O 規格をサポートします。対応する I/O 規格は、1.2 V、1.5 V、1.8 V、2.5 V、および 3.3 V のバンク電圧に対応しています。IGLOO2 MSIO 0 は 32 本の MSIO ラインを提供し、IGLOO2 MSIO 1 は 16 本の MSIO ラインを提供します。

IGLOO2 MSIO 0/1 を使用するには、キャリアボードから M2GL_VDDI8 および M2GL_VDDI5 を介してバンク電源を供給する必要があります。これにより、ユーザは IGLOO2 がサポートする任意のバンク電圧を選択できます。
